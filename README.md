# Image Processing with Pillow

This project implements a comprehensive range of **Image Processing** techniques using the **Pillow (PIL)** library. It serves as a practical guide to handling digital images natively in Python, focusing on enhancement, filtering, and data manipulation.

## 🚀 Overview
The goal of this project is to showcase the versatility of the Pillow library for digital image processing. By leveraging Pillow's specialized modules, the project demonstrates how to perform complex image analysis and modifications with efficient, high-level Python code.

## Key Features
* **Image Enhancement:** Utilizing the `ImageEnhance` module to adjust brightness, contrast, color balance, and sharpness.
* **Filter Application:** Using built-in kernels for blurring, contouring, and detail enhancement.
* **Color Level Manipulation:** Converting images between different modes (L, RGB, CMYK) and performing palette adjustments.
* **Metadata Handling:** Accessing and managing image properties such as size, format, and EXIF data.
* **Batch Processing:** Logic structures for applying transformations across multiple image files simultaneously.

## 🛠️ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/KangKang0909/Image-processing-with-Pillow
   cd Image-processing-with-Pillow
   ```

2. **Install dependencies:**
   Ensure you have the Pillow library installed:
   ```bash
   pip install Pillow matplotlib
   ```
